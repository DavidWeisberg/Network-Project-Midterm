To run the ProxyServer, execute the following make commands:

Compilation on Linux   : `make compile-linux`
Compilation on Windows : `make compile-windows`

Running the Proxy:
On Linux   : `make run-linux`
On Windows : `make run-windows`

Compilation and Running depends on JAR files present in the lib/ directory.
